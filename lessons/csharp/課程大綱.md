# C# 後端開發課程大綱

> **核心理念**：用 C# 和 .NET 打造正式級的後端服務。
> 這門課假設你**沒寫過 C#**，從語言基礎開始，一步步帶你學會物件導向、ASP.NET Core、資料庫、認證、測試，最後能獨立開發、測試、部署一個完整的後端 API。

---

## 這門課的定位

| | 說明 |
|---|------|
| **適合對象** | 想學一門「強型別、企業級」後端語言的人；想進 .NET 生態（很多企業、金融、遊戲後端在用）的人。**沒寫過 C# 也能從頭學起。** |
| **目標** | 學完能用 C# / ASP.NET Core **獨立開發一個完整的後端 API**：分層架構、接資料庫、做認證、寫測試、容器化並部署上線。 |
| **建議（非必須）** | 有 **basic 課程**的程式基礎（變數、函式、OOP、HTTP 概念）會學得更快——C# 和你學過的 TypeScript 很像（都是強型別）。沒有的話，課程也會從零解釋。 |
| **你需要準備** | 一台能裝 .NET SDK 的電腦（Windows / Mac / Linux 都行，含你的 WSL）。Part 0 會帶你裝好。 |

> 通用知識（HTTP、資料庫、SOLID、測試、Clean Code）放在頂層 `課外讀物/`，這門課會在需要時引用，不重複造輪子。
> 標 🔧 的是**動手做**章節，標 🏆 的是**整合專案**。

> **這門課和其他書怎麼搭**：basic 教你用 TypeScript/Express 做後端；這門課教你用「另一條主流路線」**C# / ASP.NET Core** 做後端。學完，部署可接 **infra**（自架）或 **aws**（雲端），可靠性可接 **sre**。

---

## 學習路徑總覽

```
Part 0 環境準備 → Part 1 C# 語言基礎 → Part 2 物件導向
   → Part 3 C# 進階特性（LINQ / async）→ Part 4 ASP.NET Core 入門
   → Part 5 建立 REST API → Part 6 資料存取（EF Core）
   → Part 7 認證與授權 → Part 8 測試
   → Part 9 工程實務與架構 → Part 10 部署上線 + 總整理專案
```

> 前半（Part 1–3）打好**語言基礎**，後半（Part 4–10）進入**後端開發實戰**。每個 Part 都有動手練習，最後整合成一個完整的後端服務。
