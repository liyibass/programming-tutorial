# AWS 雲端基礎建設課程大綱

> **核心理念**：你在公司聽到 VPC、EKS、T3 這些詞，這門課就是為此而生的。
> 從「雲端到底是什麼」開始，假設你**完全沒碰過 AWS**，一步一步帶你親手操作，最後你能看懂公司的架構圖、自己把一個網站部署上雲、也知道怎麼不被帳單嚇到。

---

## 這門課的定位

| | 說明 |
|---|------|
| **適合對象** | 想了解雲端、看懂公司架構圖、或想往 DevOps / SRE 發展的人。**完全沒用過 AWS 也沒關係**，我們從零開始。 |
| **建議（非必須）** | 對「伺服器」「網路」有基本概念會更好上手。沒有的話，課程會在需要時補充，或引導你去看 `課外讀物/`（E-1 終端機、E-3 網路、E-13 規模化）。 |
| **你需要準備** | 一個 AWS 帳號（課程第一章就教你安全地開好它）、一台能上網的電腦。**有自己的 Linux 伺服器更好**——很多概念可以兩邊對照。 |
| **學完你能** | 安全地使用 AWS、讀懂 VPC / EKS 架構圖、親手把 app 部署上雲、看懂帳單、用 IaC 管理基礎設施。 |

> **關於花費**：這門課大量使用 AWS 免費方案（Free Tier），動手練習盡量控制在免費額度內。**第一章就會教你設好預算警示，超過 1 美元就寄信通知你**，學習過程不用提心吊膽。

---

## 學習路徑總覽

```
Part 1 雲端基礎 + 安全起步（先設好防爆帳的安全網）
   → Part 2 IAM 權限 → Part 3 EC2 租一台電腦 → Part 4 VPC 私有網路（核心）
   → Part 5 儲存 → Part 6 受管服務 → Part 7 容器平台 ECS/EKS
   → Part 8 Serverless → Part 9 CI/CD 與 IaC → Part 10 觀測、成本與總整理
```

> Part 4（VPC）是理解「公司雲端架構」最核心的一章，後面的容器、受管服務都建立在它之上。
> 標 🔧 的是**動手做**章節，標 🏆 的是**整合專案**。

## Part 1 — 雲端是什麼？先安全地站上起跑線

### 章節列表

- `aws-1-1` 雲端計算的誕生：為什麼公司不再自己買伺服器了（用「租房 vs 買房」來理解）
- `aws-1-2` AWS 是一間什麼樣的店？Region / Availability Zone / Edge Location 全球架構圖解
- `aws-1-3` 先搞懂錢：免費方案 Free Tier、計費模式（On-demand / Reserved / Spot / Savings Plans）、為什麼有人會收到天價帳單
- `aws-1-4` 🔧 動手做：安全地開好你的帳號——根帳號上鎖、開啟 MFA、建立 IAM 管理者、**設定預算警示（防爆帳第一步）**
- `aws-1-5` 🔧 動手做：10 分鐘的第一個成就——用 S3 把一個網頁公開到全世界

## Part 2 — 身份與權限 IAM：誰能做什麼

### 章節列表

- `aws-2-1` IAM 是什麼？用「公司門禁卡」理解 User / Group / Role / Policy
- `aws-2-2` 最小權限原則：為什麼每個服務都要有自己的 Role，而不是全部給最大權限
- `aws-2-3` 🔧 動手做：讀懂並寫一份 IAM Policy（JSON 政策文件逐行解析）

---

## Part 3 — 計算資源 EC2：在雲端租一台電腦

### 章節列表

- `aws-3-1` EC2 是什麼？在雲端租一台電腦的完整流程
- `aws-3-2` 🔧 動手做：開一台 EC2、用 SSH 連進去、架起一個 web server（你的 Linux 經驗在這裡派上用場）
- `aws-3-3` 執行個體家族解析：**T3 / M7g / C7g / R7g** 怎麼選，Burstable Credit 機制是什麼
- `aws-3-4` AMI 與 Auto Scaling：複製一台機器 vs 根據流量自動調整台數

## Part 4 — 網路 VPC：雲端裡的私有網路（核心）

> 這是理解「公司雲端架構」最核心的章節。前面的 EC2、後面的容器與資料庫，全都住在 VPC 裡。

### 章節列表

- `aws-4-1` VPC 是什麼？為什麼需要自己的私有網路（用「封閉社區 + 門禁」來理解）
- `aws-4-2` CIDR 是什麼？`10.0.0.0/16` 這串數字代表多少 IP 位址
- `aws-4-3` 公開子網路 vs 私有子網路：什麼資源該「躲起來」
- `aws-4-4` Internet Gateway vs NAT Gateway：誰能主動出門？誰能被外部拜訪？
- `aws-4-5` Security Group vs NACL：Instance 層 vs Subnet 層的防火牆差在哪
- `aws-4-6` Route Table：封包怎麼決定要走哪條路（圖解每一跳）
- `aws-4-7` Multi-AZ 架構：為什麼要把資源分散到不同機房
- `aws-4-8` 🔧 動手做：親手建一個 VPC，把 Part 3 的 EC2 放進私有子網路
- `aws-4-9` 看懂 VPC 架構圖：從一張圖讀懂公司的雲端網路全貌

---

## Part 5 — 儲存

### 章節列表

- `aws-5-1` S3 物件儲存：概念、Bucket Policy、使用場景（回頭看懂 `aws-1-5` 做了什麼）
- `aws-5-2` EBS vs EFS：Block Storage（本機硬碟）vs Shared File System 的差別
- `aws-5-3` S3 進階：Versioning / Lifecycle Policy / Presigned URL
- `aws-5-4` 🔧 動手做：用 S3 + Presigned URL 做一個「安全上傳檔案」的功能

---

## Part 6 — 受管服務：讓 AWS 幫你顧

### 章節列表

- `aws-6-1` 為什麼要用「受管服務」？自己架 vs 給 AWS 顧的取捨
- `aws-6-2` RDS：托管關聯式資料庫 — Multi-AZ vs Read Replica 差在哪
- `aws-6-3` ElastiCache：AWS 上的受管 Redis，跟自己架有什麼不同
- `aws-6-4` Application Load Balancer（ALB）：流量怎麼分配、健康檢查怎麼運作
- `aws-6-5` CloudFront CDN：前端靜態資源部署加速的標準做法（前端工程師必看）
- `aws-6-6` Route 53 + ACM：設定自訂網域與 HTTPS 憑證的完整流程

---

## Part 7 — 容器平台 ECS / EKS

### 章節列表

- `aws-7-1` 為什麼要容器？從「直接跑在 EC2」到容器化的演進（先看 `課外讀物/` Docker 基礎）
- `aws-7-2` ECR：Docker Image 的倉庫（AWS 版的 Docker Hub）
- `aws-7-3` ECS vs EKS：AWS 提供的兩條容器路線，各自適合什麼
- `aws-7-4` 🔧 動手做：把一個 Docker app 部署到 ECS Fargate（最平易近人的容器入門，不用管底層機器）
- `aws-7-5` **EKS 架構圖解**：Control Plane / Node Group / Fargate / kubelet 各司其職
- `aws-7-6` **EKS 網路深入**：Pod IP 從哪來？Service / Ingress / CoreDNS 怎麼串起來
- `aws-7-7` HPA 與 Cluster Autoscaler：流量衝高時 Kubernetes 如何自動橫向擴容
- `aws-7-8` Helm Chart：Kubernetes 的套件管理員，讀懂 `values.yaml`
- `aws-7-9` 看懂公司的 EKS 架構：真實場景全景拆解（結合 VPC + ALB + RDS）

## Part 8 — Serverless

### 章節列表

- `aws-8-1` Lambda 是什麼？函數即服務（FaaS）的概念與冷啟動問題
- `aws-8-2` 🔧 動手做：API Gateway + Lambda 做一個無伺服器 API（不用 Express 也能有 HTTP API）
- `aws-8-3` 選型指南：Lambda vs ECS vs EKS vs EC2 — 什麼場景用什麼

## Part 9 — CI/CD 與基礎設施即代碼 IaC

### 章節列表

- `aws-9-1` GitHub Actions + AWS：程式碼推上去自動部署到 ECS / EKS
- `aws-9-2` 為什麼不能一直點 Console？Infrastructure as Code 的必要性
- `aws-9-3` 🔧 動手做：Terraform 入門——把 Part 4 的 VPC 改寫成可版本控制的程式碼
- `aws-9-4` Terraform State：多人協作時誰的狀態才是對的？
- `aws-9-5` CloudFormation 與 CDK：AWS 原生的 IaC 選項，跟 Terraform 有什麼不同

## Part 10 — 觀測、除錯與成本

### 章節列表

- `aws-10-1` CloudWatch：Log / Metrics / Alarm — 線上出問題時怎麼找線索
- `aws-10-2` AWS X-Ray：分散式追蹤，找出哪個服務拖慢了整體回應
- `aws-10-3` Cost Explorer 與 Budgets：看懂帳單、設定花費上限、避免意外爆帳
- `aws-10-4` 趣味：AWS 大當機事件 — 一個打錯的 CLI 指令讓半個網路斷線兩小時
- `aws-10-5` 🏆 總整理專案：把一個真實 app 完整部署上 AWS（VPC + ECS Fargate + RDS + ALB + CloudFront + 自訂網域），並設好監控與預算
