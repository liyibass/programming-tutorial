## 概念說明

### 函式 vs 巨集：一個直覺的分法

你已經很熟函式了（[rust-1-4]）。Rust 還有一種東西叫**巨集**，呼叫時名字後面跟著 **`!`**，例如 `println!`、`vec!`。它和函式有什麼不同？

一個夠用的直覺：

```
函式：執行時被呼叫，接受固定數量、固定型別的參數。
巨集：在「編譯前」就先「展開」成更多程式碼，
      可以接受「數量不定」的參數、做函式做不到的事。
```

比喻一下：函式像「打一通電話請人幫你做一件事」；巨集像「一個會自動幫你把草稿擴寫成完整文章的模板」——在程式真正編譯之前，巨集就先把自己「展開」成一段對應的程式碼。

為什麼 `println!` 要是巨集？因為它要能接受**任意數量的參數**：

```rust
println!("只有一句");
println!("{} 加 {} 等於 {}", 1, 2, 3);   // 參數數量隨你
```

一般函式很難優雅地「參數數量隨便變」，巨集做得到。所以「印出東西」這種需要彈性的工作，Rust 用巨集實作。**現階段你只要記住：看到 `!` 就是巨集，照著用即可**，怎麼自己寫巨集是很後面的進階主題。

## 程式碼範例

### `{}` 佔位符：把值填進字串

`println!` 用 `{}` 當「佔位符」，後面依序給值：

```rust
fn main() {
    let name = "小美";
    let age = 28;
    println!("{} 今年 {} 歲", name, age);   // 小美 今年 28 歲
}
```

說明：第一個 `{}` 填 `name`、第二個填 `age`，**依順序對應**。

也可以在 `{}` 裡寫變數名，更好讀（Rust 較新版本支援）：

```rust
fn main() {
    let name = "小美";
    let age = 28;
    println!("{name} 今年 {age} 歲");   // 直接把變數名寫進去
}
```

### `{:?}` ：印出「給開發者看」的格式

當你想印出比較複雜的東西（像 tuple、之後的 struct）來除錯，普通 `{}` 可能不夠，要用 `{:?}`（debug 格式）：

```rust
fn main() {
    let point = (3, 5);
    println!("{:?}", point);    // (3, 5)
}
```

`{:?}` 是除錯好幫手。之後 [rust-5-5] 你會學到，自訂的型別只要加一行 `#[derive(Debug)]` 就能用 `{:?}` 印出來。

### 幾個你會常用的巨集

| 巨集 | 做什麼 |
|------|--------|
| `println!` | 印一行（自動換行） |
| `print!` | 印出但不換行 |
| `format!` | 不印出，而是「組出一個字串」回傳 |
| `vec!` | 快速建立一個 `Vec`（[rust-6-1] 會用到） |
| `panic!` | 讓程式立刻中止並報錯（[rust-4-1] 會講） |

例如 `format!` 很實用——它和 `println!` 用法一樣，但不是印出來，而是把結果**當成字串給你**：

```rust
fn main() {
    let greeting = format!("Hello, {}!", "Rust");
    println!("{}", greeting);   // greeting 是一個 String
}
```

## 小練習

1. 用一個 `println!` 印出「我叫 ___，我在學 Rust」，把名字用變數帶入（試試 `{name}` 直接寫變數名的寫法）。
2. 用 `{:?}` 印出一個 tuple `(身高, 體重)`。
3. 用 `format!` 把「年」「月」「日」三個數字組成一個字串 `2026-06-24` 的樣子（提示：`format!("{}-{}-{}", y, m, d)`），存到變數再印出。
