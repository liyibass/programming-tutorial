建立新的課程章節。

使用方式：/new-lesson [章節編號] [標題]
例如：/new-lesson 2-3 複合型別：object / array / tuple

步驟：
1. 根據章節編號判斷放在哪個 `lessons/part-X/` 資料夾
2. 檔名格式：`{編號}-{標題slug}.md`，例如 `2-3-compound-types.md`
3. 用以下模板建立檔案，並根據標題填入對應內容

模板：
```markdown
# [{編號}] {標題}

> **本章目標**：（一句話說明學完這章能做什麼或理解什麼）

## 你會學到

-
-
-

## 概念說明

（先用 pseudo code 或日常類比描述概念，再帶出正式說明）

## 程式碼範例

（每段程式碼前說明它在做什麼）

## 小練習

1.
2.

## 課外讀物

> （如有對應的課外讀物，用以下格式引用，否則刪除此區塊）
> 想深入了解 XXX → **[課外讀物 E-X-X] 完整標題**
```

建立完成後，輸出檔案路徑，並提示使用者接著填寫「本章目標」和「概念說明」。
