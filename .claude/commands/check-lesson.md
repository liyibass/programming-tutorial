審查指定章節是否符合 CLAUDE.md 的規範。

使用方式：/check-lesson [檔案路徑]
例如：/check-lesson lessons/part-2/2-3-compound-types.md

審查步驟：
1. 讀取指定的 .md 檔案
2. 逐項對照以下 checklist，標記通過 ✅ 或未通過 ❌

Checklist：
- [ ] 有「本章目標」（一句話說明）
- [ ] 概念說明遵循「類比/pseudo code → 圖表 → 程式碼」順序
- [ ] 抽象概念有 pseudo code 或生活類比先行
- [ ] 適當使用 Mermaid 圖表（如果該章節有流程、結構、時序需要說明）
- [ ] 有程式碼範例，且每段範例前有說明文字
- [ ] 有「小練習」區塊（課外讀物章節可免）
- [ ] 如有對應課外讀物，有正確格式的引用
- [ ] 如是 POC 解鎖章節，有 POC 引導區塊
- [ ] 沒有連續超過 30 行未加說明的程式碼
- [ ] 術語第一次出現時有解釋

輸出格式：
- 列出每個項目的通過狀態
- ❌ 的項目附上具體說明（缺什麼、在哪裡補）
- 最後給一個總結：「可以發布」或「需要修改 N 項」
